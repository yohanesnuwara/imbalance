# imbalance
Notebooks on various machine learning and neural networks methods for imbalanced classification

Almost every phenomenon in geoscience is an extremely imbalanced distribution. For example, the distribution of trace minerals in the drill cuttings (Nuwara et al, 2026). For the drill cuttings case, even an instrument called X-Ray Diffraction (XRD) is not able to detect it when below threshold. Modern ML and NN approaches is successful in solving ideal cases, however they struggle when the labels are extremely imbalanced and noisy. This problem has been known for so many years, but only a few is really putting an emphasis on "understanding" it and solve the problem. 

This repo is a quest of my exploration of imbalanced data modelling, with some deep dive in the maths, algorithms, and code. 

## Spotlight on some researches

There are some notable approaches for extreme imbalanced classification:

* Self paced ensemble 
* Sample learning pair
* Cost sensitive neural netwok
* Imbalanced XGBOOST
